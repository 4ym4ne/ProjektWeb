Der Spread-Operator in JavaScript

Der Spread-Operator (...) ermöglicht es, ein iterierbares Objekt wie ein Array oder eine Zeichenkette in Elemente/Argumente zu expandieren, die in Funktionsaufrufen verwendet oder in Arrays kopiert werden können. Er kann auch verwendet werden, um Objekteigenschaften in ein neues Objekt zu kopieren.

Beispielverwendung

Ein einfaches Beispiel für die Verwendung des Spread-Operators zum Verbinden von Arrays:


    // Definieren zweier Arrays
    const array1 = [1, 2, 3];
    const array2 = [4, 5, 6];

    // Verwenden des Spread-Operators zum Verbinden der Arrays
    const kombiniertesArray = [...array1, ...array2];

    console.log(kombiniertesArray); // [1, 2, 3, 4, 5, 6]

Dieser Operator ist besonders nützlich, wenn man mit Funktionen arbeitet, die eine Vielzahl von Argumenten erwarten oder wenn man schnell Kopien von Arrays/Objekten erstellen möchte, ohne die Originaldaten zu verändern.
